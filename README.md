## ‚ú® Features

*   **Entropy Calculator**: Compute Shannon entropy for discrete random variables, a cornerstone of information theory.
*   **Huffman Coding**: Implementation of the classic lossless data compression algorithm, demonstrating efficient data representation.
*   **Channel Simulation**: Models for Binary Symmetric Channel (BSC) and Binary Erasure Channel (BEC) to analyze noisy communication.
*   **Joint & Conditional Entropy**: Tools to analyze relationships and dependencies between multiple random variables.

## üöÄ Tech Stack

*   **Matplotlib**: Used for data visualization, plotting probability distributions, and channel characteristics.
*   **SciPy**: Provides additional scientific computing tools and algorithms.
